Text Button
    ลักษณะ : ปุ่มที่มีลักษณะเป็นเส้นขอบมีสี ไม่มีสีแบ็คกราวน์
    วิธีการใช้งาน : เหมาะกับ Action ที่มีความสำคัญรองลงมาเป็นทางเลือกที่สอง หรือทางเลือกการปฏิเสธเช่น ยกเลิก

ขั้อตอนการทำงานของการเปลี่ยนหน้า Navigator.Push
    สร้าง Method Navigator.push ขึ้นมา และจำเป็นต้องส่งค่า BuildContext กับ widget

    BuildContext สถานะการทำงาน (ไม่เก็ทฮะ)
    Widget หน้าที่ต้องการจะเปลี่ยนแทนค่าว่า Page (Class ของหน้าที่ต้องการนำมาแสดงผล)

    ทำงานร่วมกับปุ่มต่างๆ 
    ในปุ่มจะมีคำสั่ง onPressed ให้เราสั่งให้ Method Navigator ทำงานโดยส่งค่า context กับ (ชื่อคลาสของหน้าที่ต้องการ) เช่น TextPage()

    สิ่งที่มันทำไม่ใช่การเปลี่ยนหน้า แต่เป็นการเอาหน้าใหม่ขึ้นมาบังแทนหน้าเก่ายกตัวอย่างเช่น
    
    ตอนสั่งรัน หน้า Main เปิด Homepage ขึ้นมา
    Homepage index -> 1
    เมื่อสั่ง Navigator ให้เปิด TextPage() ขึ้นมา

    Textpage index -> 2
    Homepage index -> 1

    จึงเป็นการแสดงผลว่า  Textpage อยู่ข้างบน Homepage ในมุมมองแบบ Layer

    ทำงานร่วมกับ navigator.pop คือทำลาย Index ปัจจุบัน จากการทำราย Index 2 ก็จะเหลือแค่ Index 1 ซึ่งเป็นหน้าก่อนหน้า
